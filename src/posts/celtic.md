# Celtic FC
# Селтик
**Селтик**  (англ. The Celtic Football Club) — шотландский профессиональный футбольный клуб из города Глазго. Выступает в шотландском Премьершипе, высшем дивизионе Шотландской профессиональной футбольной лиги.
